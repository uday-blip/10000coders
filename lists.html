Frontend :

The frontend is the tier which contains all the user interface and user experience , in simple words it is the visible part that is appearing to the user.

Backend :

The backend is the tier that contains all the logic and runs the bussiness by taking the help of database , it simply gives response to the user requests by applying some real time logic to the data in the database
