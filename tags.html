doctype html : it describes the docttype and html version.
html: root element of the html webpage
head: contains the meta information about the documents
title: sets the title to the webpage
h1 to h6: defines headings high to small size
div:A generic container for grouping and styling content.
footer : Defines the footer section of a webpage
p : defines a paragraph of text.
b : simply makes text bold.
br : inserts a line break (no closing tag needed).
hr :Creates a horizontal rule(a thematic break).
a : defines a hyperlink.
img : embeds video and audio files
List tags : ul : defines an unordered (bulleted) list.
ol : defines an ordered (numbered) list.
li: Represents a list item inside ul and ol tag.
Form tags:
input tag:Defines an input field (text,password,checkbox,etc).
button: creates a clickable buton.
select: creates a dropdown list.
Semantic Tags in html :
Section : Defines a section in a document (eg. chapters ,headers).
Article : Represents independent ,self-contained content(eg, blog spot).
Nav: Contains navigation links.
Aside: Defines a content aside from the main content (eg, sidebar).
Table tags:
Table : Creates a Table
Tr: Defines a table row.
Th: Defines a table data cell.
Td: Defines a table data cell.
marquee : This tag is used to create a motion to the text or img in our webpage in the direction we want.
meta tag: Provides a metadata about the html document (eg charset, viewport, description).
Link tag: Links external resources like Css files, Favicons or preloads.

So these are the Only frequently used tags in html
